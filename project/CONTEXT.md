# サイト概要
- サイト名：AI学習ナビ for Parents
- ターゲット：小中学生の子を持つ保護者
- コンセプト：AIを怖いものではなく「学習の味方」として紹介

# ターゲットが抱える悩み
- 子どもにAIを使わせていいか不安
- どのAIツールが子どもに向いているかわからない
- AI時代に何を勉強させればいいかわからない
- 通信教育とAIをどう組み合わせるか知りたい

# 扱うアフィリエイト案件（候補）
- 通信教育（進研ゼミ、Z会、スマイルゼミなど）1,000〜5,000円/件
- AIを使った学習アプリ・サービス
- プログラミング教室・オンライン学習（1〜3万円/件）
- 教育系書籍（Amazon）
- タブレット・学習用デバイス

# コンテンツ方針
- 保護者目線で安心・信頼感のある文体
- 体験談・比較記事・使い方ガイドを中心に
- 難しいAI用語は使わず、わかりやすく説明する
